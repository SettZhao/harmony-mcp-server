# usb-serial-for-android 三方库规格

## 1. 库概述

**原始库**: usb-serial-for-android  
**原始平台**: Android  
**目标平台**: HarmonyOS API 12+ (NEXT)  
**功能**: 通过 USB 主机模式与 USB 转串口设备通信，支持 FTDI、CDC/ACM、CP21xx、Prolific (PL2303)、CH34x 芯片。

## 3. Android → HarmonyOS 接口映射总览

| Android 接口 | HarmonyOS 接口 | 说明 |
|-------------|---------------|------|
| `UsbManager.getDeviceList()` | `usbManager.getDevices()` | 返回 `Array<usbManager.USBDevice>` |
| `UsbManager.openDevice(device)` → `UsbDeviceConnection` | `usbManager.connectDevice(device)` → `USBDevicePipe` | 需先调用 `requestRight` |
| `UsbDeviceConnection.claimInterface(iface, force)` | `usbManager.claimInterface(pipe, iface, force)` | 同步，返回 0 表示成功 |
| `UsbDeviceConnection.releaseInterface(iface)` | `usbManager.releaseInterface(pipe, iface)` | 同步 |
| `UsbDeviceConnection.bulkTransfer(ep, buf, len, timeout)` | `usbManager.bulkTransfer(pipe, ep, buf, timeout)` → `Promise<number>` | **异步** |
| `UsbDeviceConnection.controlTransfer(reqType, req, val, idx, buf, len, timeout)` | `usbManager.usbControlTransfer(pipe, {bmRequestType, bRequest, wValue, wIndex, wLength, data}, timeout)` → `Promise<number>` | **异步** |
| `UsbDeviceConnection.close()` | `usbManager.closePipe(pipe)` | 同步 |
| `UsbDevice.getVendorId()` | `device.vendorId` | 属性直接访问 |
| `UsbDevice.getProductId()` | `device.productId` | 属性直接访问 |
| `UsbDevice.getInterfaceCount()` | `device.configs[0].interfaces.length` | 通过配置描述符 |
| `UsbDevice.getInterface(i)` | `device.configs[0].interfaces[i]` | |
| `UsbInterface.getEndpointCount()` | `iface.endpoints.length` | |
| `UsbInterface.getEndpoint(i)` | `iface.endpoints[i]` | |
| `UsbEndpoint.getType()` | `ep.type` | `2` = BULK |
| `UsbEndpoint.getDirection()` | `ep.direction` | `0x80` = IN, `0` = OUT |
| `UsbConstants.USB_DIR_IN` | `0x80` | 常量 |
| `UsbConstants.USB_DIR_OUT` | `0x00` | 常量 |
| `UsbConstants.USB_TYPE_CLASS` | `0x20` | 常量 |
| `UsbConstants.USB_TYPE_VENDOR` | `0x40` | 常量 |
| `UsbConstants.USB_ENDPOINT_XFER_BULK` | `2` | 常量 |
| `byte[]` | `Uint8Array` | 字节数组类型 |
| `Thread` / `Runnable` | `async/await` + 异步循环 | HarmonyOS 无线程模型 |
| 反射 `Class<Driver>` | 工厂函数 `DriverFactory` | ArkTS 不支持运行时反射 |
| `synchronized` 块 | 移除（JS 单线程） | |

## 4. 不支持特性说明

| Android 特性 | 处理方式 |
|-------------|---------|
| `UsbRequest` 异步读取 | HarmonyOS `bulkTransfer` 本身是 Promise 异步，无需 `UsbRequest` |
| Prolific 状态线程 (`mReadStatusThread`) | 使用异步轮询替代（通过 `setInterval` 或循环 async） |
| 反射实例化驱动 (`Constructor.newInstance`) | 改为工厂函数模式 |
| `synchronized` 关键字 | 移除（ArkTS/JS 单线程，无需同步原语） |
| `System.arraycopy` | 改为 `Uint8Array.set()` |
| `ByteBuffer` | 改为直接操作 `Uint8Array` |
| `byte` 类型（有符号 -128~127） | 使用 `number & 0xFF` 保证 8 位无符号语义 |
